# Customer Segmentation using Machine Learning

## Overview

This project segments customers based on purchasing behavior using RFM analysis and clustering algorithms.

## Algorithms Used

* K-Means
* Hierarchical Clustering
* DBSCAN
* Gaussian Mixture Model

## Steps

1. Data Cleaning
2. Feature Engineering (RFM)
3. Scaling
4. Clustering
5. Visualization

## Results

* High-value customers → target with premium offers
* Low-value customers → discounts
* At-risk customers → re-engagement strategies

## How to Run

```
pip install -r requirements.txt
python src/main.py
```
